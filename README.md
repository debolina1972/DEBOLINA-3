# Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project uses K-Means Clustering to segment customers based on purchasing behavior and spending patterns.

The system helps businesses improve marketing strategies and customer targeting.

---

## 🎯 Objectives
- Group customers into meaningful segments
- Identify high-value customers
- Improve marketing strategies
- Increase business profitability

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

## 📂 Dataset Features
- Customer ID
- Age
- Gender
- Annual Income
- Spending Score
- Purchase Frequency

---

## ⚙️ Algorithms Used
- K-Means Clustering
- Elbow Method

---

## 🚀 Project Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Elbow Method
5. K-Means Clustering
6. Visualization
7. Cluster Analysis

## 📊 Expected Output
- Customer clusters

- Visual segmentation graphs
